Dynamic Snippets Builder

Easily create dynamic snippets with desired data in seconds!

The Dynamic Snippets Builder module enables you to create dynamic snippets with the data you need without undergoing a full development process.


This module provides a simple page to specify your needs:
  • The model to extract data from
  • Desired fields
  • Theme (Card or List)
  • Filter data
  • Snippet visibility for end users

Accessing the Module: Navigate to Website -> Configuration -> Snippets.

Form View: Specify required data like model, fields, theme, filter, and access group.

After creation, find your snippet in the Website Builder Menu.
